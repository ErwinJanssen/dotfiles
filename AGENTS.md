# Agent Instructions

## Repo Type
Nix Flake + Home Manager configuration for personal dotfiles.

## Key Commands
- `nix run home-manager -- switch --flake .` - Apply configuration
- `make theme.json` - Generate theme colors from scripts/theme.py
- `git clean -xdf` - Clean generated files

## Structure
- `flake.nix` - Root flake with inputs (nixpkgs, home-manager, nixgl)
- `nix/home-manager/` - Home Manager modules
- `scripts/theme.py` - Python script generating theme.json (requires colormath)
- Generated files: `theme.json` (in .gitignore)

## Toolchain
- Nix Flakes enabled (`nix-command`, `flakes`)
- Pre-commit: ruff (Python), stylua (Lua), nix fmt, fish_indent
- ruff: `select = ["ALL"]` with per-file ignores for scripts/*
- stylua: `no_call_parentheses = true`, spaces for indent

## Gotchas
- theme.json is generated by Makefile from scripts/theme.py - do not edit directly
- Home Manager stateVersion: 23.11
- Nixpkgs follows nixos-unstable
- nixGL used for GPU acceleration with defaultWrapper = "mesa"
